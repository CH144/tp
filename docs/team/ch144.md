### Project: Big Brother

Big Brother is a desktop contact management application used by Human Resources of startups to manage a relatively small number of employees' information (around 50). The user interacts with it using a CLI, and it has a GUI created with JavaFX. It is written in Java, and has about 10 kLoC.

Given below are my contributions to the project.

* **New Feature**: Added auto optimization of number of displayed columns according to window width.
  * What it does: allows the user to view as many contacts as reasonably possible on a single screen.
  * Justification: this features improves the browsability of the application as the user does not need to keep scrolling to view more data.
  * Credits: ChatGPT was used to help with the usage of JavaFx classes and functions.
